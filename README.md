## Inatallation

Clone the repository
```
git clone https://github.com/Sakshikulkarni880/Employee-Registration-Frontend.git
```

Install dependencies
```
cd employee-registration
cd frontend
cd client
npm install
```

### Start the development server
```
npm start
```
